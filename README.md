# Bot Agent HTML Guide

Локальный проектный пакет с гидом по созданию бота/агента и образцом prompt.

## Состав

- `index.html` — основной HTML-гид
- `BOT_CREATION_PROMPT.md` — образец prompt для создания бота
- `GITHUB_STATUS.md` — статус публикации и что нужно для переноса в GitHub

## Для чего это

Этот пакет нужен как:
- понятная база для создания onsite-бота или агента;
- краткий гид без перегруза документацией;
- шаблон для дальнейшей публикации в GitHub;
- референс для рабочих обсуждений по сайту, чату, подборщику и каналу обращений.

## Что важно

- Telegram не закладывается как единственная обязательная опора;
- основной сценарий должен жить внутри сайта или внутри контролируемого продукта;
- любые prompt- и agent-решения должны сначала опираться на подтвержденные материалы.
