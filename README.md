# TEC

Репозиторий используется как база и для HTML-гида по ботам/агентам, и для будущего сайта/проекта.

## Что уже есть

- `index.html` — стартовый HTML-гид по созданию бота или агента
- `BOT_CREATION_PROMPT.md` — образец prompt
- `GITHUB_STATUS.md` — первоначальный статус публикации

## Дерево проекта

- `guide/` — guide-материалы и дублирующая точка входа
- `site/` — база под сам сайт
- `site/content/` — подтвержденные текстовые данные и рабочие сводки
- `site/assets/images/` — изображения для сайта
- `site/assets/docs/` — PDF, datasheet, спецификации
- `prompts/` — prompt-образцы
- `sync/` — локальная схема синхронизации

## Важные правила

- не строить критический сценарий только на Telegram;
- сначала подтвержденные источники, потом AI;
- терминология, модели и характеристики должны сверяться с исходниками.
