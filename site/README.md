# Site

Эта папка зарезервирована под сам сайт/проект.

## Структура

- `content/` — тексты, сводки, структура моделей, рабочие данные
- `assets/images/` — изображения
- `assets/docs/` — PDF и документация

## Принцип

Сначала сюда собираются подтвержденные материалы, потом на их основе строится сам сайт.
