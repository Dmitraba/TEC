# Channel Options

Варианты каналов обращения для сайта.

## Базовое правило

Telegram не использовать как единственный основной канал сайта.

## Варианты

1. Onsite-чат
2. AI-консультант по базе сайта
3. Пошаговый подборщик
4. Гибрид: чат + форма
5. Простая форма / callback-виджет

## Рекомендация

Для технического B2B-сайта лучше всего:
- основной канал: onsite-чат или AI-консультант
- параллельно: форма подбора / запроса КП
- Telegram: только дополнительный маршрут
